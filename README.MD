# Trabalho de Extensão

Este repositório foi criado para armazenar a documentação e os artefatos relacionados ao meu Trabalho de Extensão da faculdade.

No momento, o repositório contém apenas o arquivo **`arquitetura.md`**, que apresenta um rascunho técnico da arquitetura e da estrutura geral do projeto. À medida que o desenvolvimento avançar, novos documentos e recursos serão adicionados para complementar a documentação e registrar a evolução do trabalho.
